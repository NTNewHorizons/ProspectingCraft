# ProspectingCraft

ProspectingCraft is a Minecraft mod to help you find ores. With ProspectingCraft, you can take Geological Samples and analyse them to get an idea of what kinds of ores are present in the vicinity. You can also build a Geological Surveyor to seismically map and underground area, showing which spots may contain something worth mining.

Original mod by Greg Ewing - https://www.csse.canterbury.ac.nz/greg.ewing/minecraft/mods/ProspectingCraft/
